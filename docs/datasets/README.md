# Datasets

This repository does not bundle full image datasets.

- Source-task dataset: `FETAL_PLANES_DB_data`, public at https://zenodo.org/records/3904280
- Target-task dataset: `mendeley_iqa_701515_grouped900`, derived from public releases at https://www.kaggle.com/datasets/orvile/dataset-for-fetus-framework and https://zenodo.org/records/11005384

The released metadata files document the source dataset table and the target dataset grouping/merge process.
